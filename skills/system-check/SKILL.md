# System Check Skill

Автоматическая проверка всех систем OpenClaw.

## Триггер

Фразы: `* проверка`, `проверь системы`, `системный чек`, `check systems`

## Выполняемые проверки

### 1. Сеть и интернет
- curl к example.com, google.com
- ping 8.8.8.8

### 2. web_fetch / ollama_web_search
- Тестовый запрос

### 3. Memory search
- Поиск по MEMORY.md и memory/*.md

### 4. File index
- Проверка БД
- Поиск по индексу

### 5. Gateway и плагины
- Статус процесса
- Загруженные плагины

### 6. Telegram
- Статус подключения

### 7. Discord
- Логи подключений
- WebSocket статус

### 8. Browser
- Статус CDP

### 9. Crypto skill
- Тестовый запрос курсов

### 10. Nodes
- Список сопряжённых устройств

## Формат отчёта

```
## System Check Report

| Система | Статус | Примечание |
|---------|--------|------------|
| ✅/❌/⚠️ | ... | ... |

### Критичные проблемы
- ...

### Рекомендации
- ...
```

## Примечания

- Не изменяет конфигурацию без разрешения
- Только диагностика
- Результат сохраняется в memory/YYYY-MM-DD.md при критичных ошибках
